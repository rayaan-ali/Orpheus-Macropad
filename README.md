# Orpheus Macropad

A compact, fully customizable 3-key macropad built for the Hack Club Hackpad YSWS.

## Overview

The Orpheus is my DIY macropad that features 3 hot swappable switches which can be mapped to any shortcut, a rotary encoder for hardware volume control, and an OLED display which will be fully programmable from Arduino. The designs are made in KiCad with a custom model for the mechanical structure in Fusion 360. 

## Features

- 3 hot-swappable MX mechanical switches (fully remappable)
- Rotary encoder for hardware volume control
- Customizable OLED display (Arduino-programmable)
- USB-C via Seeed Studio XIAO RP2040
- Custom PCB designed in KiCad
- Sandwich-style enclosure designed in Fusion 360
